# Plant-app
react-native, spring boot을 활용한 식물 관리 App

기능 
1. 자신이 키우는 식물을 저장합니다.
2. 키우는 식물의 물 주기 일정을 달력에 표시합니다.
3. 키우는 식물을 조회합니다.
4. 키우는 식물을 수정합니다.
5. 키우는 식물을 삭제합니다. 
6. 물 주기 일정은 현재 날씨와 물을 준 일자를 기준으로 +- 3일로 자동으로 계산됩니다.
